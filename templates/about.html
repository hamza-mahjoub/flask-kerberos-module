Welcome

Project Description : This project illustrate the kerberos authentification feature in the case of repository(folder) content protection.A user will not be able to access files within an exposed folder on the server without having a valid kerberos ticket.

Technologie Used : This project is based on the flask kerberos module.

Prerequisites : A kerberos architecture with at least 2 machines (one playing the kdc and the other a client).
